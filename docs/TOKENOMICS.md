# LastroForge Tokenomics: The Self-Backed Economy

LastroForge operates on a Zero-Fiat architecture where value is derived from Digital Scarcity, Provenance, and Utility.

## 1. Asset Classes

### Forge Artifacts (NFTs - ERC721/1155)
- **Supply**: Dynamic, based on creation fee and curation.
- **Burning Mechanism**: Users can burn artifacts to extract "Forge Dust".
- **Value Multiplier**: Value = Base + (Provenance Depth * Rep Score).

### Lastro Coins (ERC20)
- **Utility**: The "Fuel" of the forge. Used for naming, evolved metadata, and priority minting.
- **Deflation**: 2% of every marketplace transaction is burned. 100% of "Naming Fees" are burned.

## 2. The Provenance Engine
Unlike traditional NFTs where value is static, LastroForge introduces **Accumulative Brand Value**.
- Whenever an artifact is traded, a percentage of the value is "staked" into the artifact itself.
- This creates a floor price that grows with every transfer.

## 3. Launch Strategy (2026-2027)
- **Phase 0: The Alpha Forge**: Invitation-only forging for key cultural creators.
- **Phase 1: Genesis Airdrop**: Artifact shards distributed to active Web3 residents.
- **Phase 2: Viral Loop**: "Forge Referral" system where referrers get a "Legacy Seal" on artifacts created by their recruits.
